# Simple Coin Collector

A small 2D platformer prototype built in Unity 6 to demonstrate core gameplay mechanics.

## Features
- Player movement and jumping with Rigidbody2D physics
- Multiple collectible coins with trigger detection
- Real-time score system
- Win condition and restart functionality
- Camera follows player
- Proper Unity 6 input handling setup

## Controls
- Move: A/D or Arrow Keys
- Jump: Space

## What I Learned
- Setting up 2D physics and colliders
- Input management in Unity 6 (legacy + new system)
- Trigger events and object destruction
- Basic UI implementation
- Proper project structure and version control

## How to Run Locally
1. Clone the repo
2. Open in Unity 6
3. Set Active Input Handling to "Both" in Project Settings → Player
4. Press Play

Built as a learning project to strengthen fundamentals before moving to larger systems.
